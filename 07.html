07 配置 Nginx 实现反向代理

07 配置 Nginx 实现反向代理

7.1 配置 Nginx 反向代理 Nodejs 端口

  • sudo apt-get nginx
  • nginx -v
  • cd /etc/nginx/conf.d
  • sudo vim my-site-com-8081.conf 创建一个配置文件
upstream imooc {
  server 127.0.0.1:8081;
}

server {
  listen 80;
  server_name your.domain.com;

  location / {
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forward-For $proxy_add_x_forwarded_for;
    proxy_set_header Host $http_host;
    proxy_set_header X-Nginx-Proxy true;

    proxy_pass http://imooc;
    proxy_redirect off;
  }
}
  • sudo nginx -t
  • sudo nginx -s reload
  • 修改 basic config，隐藏版本信息：/etc/nginx/ning.conf，在 Basic Setting 中取消server_tokens off 的注释，重启 nginx 服务。
